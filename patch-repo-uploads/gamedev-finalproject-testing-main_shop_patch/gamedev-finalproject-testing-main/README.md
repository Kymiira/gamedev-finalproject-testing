# gamedev-finalproject-testing
[version index](https://kymiira.github.io/gamedev-finalproject-testing/)

Given a version number MAJOR.MINOR.PATCH, increment the:

1. MAJOR version when you make incompatible API changes<br>
2. MINOR version when you add functionality in a backwards-compatible manner<br>
3. PATCH version when you make backwards-compatible bug fixes.
